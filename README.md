# BikeMate
The iOS App for BikeMate Intelligent Lock , Based on Bluetooth4.0

“BikeMate智能车载系统”是一个基于蓝牙4.0的面向自行车骑友的综合系统，它包括硬件端的GPS定位、智能加锁、防盗报警、行车发电等功能，软件端的地图导航、蓝牙控制、丢失找回等功能。
